About

Penjelasan Aplikasi

Chord Recognition merupakan aplikasi pengenalan kunci lagu terhadap data masukan berupa file mp3 atau wav. Data masukan dapat dilakukan upload atau melakukan rekaman menggunakan aplikasi. Berikut merupakan beberapa poin penting pada aplikasi:

Tujuan Utama Aplikasi

Melakukan pengenalan data lagu yang dimasukkan oleh user.

Fitur Pengenalan: File Upload

Pengenalan akor dapat dilakukan setelah melakukan upload file dari PC anda.

Fitur Pengenalan: Audio Recording

Pengenalan akor dapat dilakukan dengan cara merekam secara langsung menggunakan perangkat keras Audio-in pengguna.

Tidak Memerlukan Instalasi

Aplikasi dapat langsung dilakukan pada laman web ini.

Aplikasi Gratis!

Chord Recognition merupakan aplikasi gratis berbasis web yang bisa anda gunakan kapanpun.

Tidak Memerlukan Registrasi

Fitur-fitur yang ada pada aplikasi ini dapat langsunug dilakukan tanpa melakukan proses registrasi dalam bentuk apapun.
